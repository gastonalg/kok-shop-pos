
# KOK Shop POS (Inventario + Ventas + Dólar del día)

Sistema multi‑dispositivo para KOK Shop:

- Productos con precio USD
- Dólar del día manual
- Stock centralizado
- Ventas por vendedor
- Reportes diarios / mensuales
- Import Excel

## 🚀 Publicar frontend (GitHub Pages)
1. Settings → Pages
2. Deploy from branch
3. Branch main / folder web

## 🧠 Backend + DB
Deploy recomendado:
- DB → Railway (MySQL)
- API → Render

## 📦 Import Excel
Subir archivo → endpoint /api/import
